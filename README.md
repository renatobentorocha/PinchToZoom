## :information_source: How To Use

```bash
# Configure expo (https://expo.io)

# Clone this repository
$ git clone https://github.com/renatobentorocha/PinchToZoom.git PinchToZoom

# Go into the repository
$ cd PinchToZoom

# Install dependencies
$ yarn

# Run the app (iOS)
$ expo start --ios

# Run the app (Android)
$ expo start --android
```
